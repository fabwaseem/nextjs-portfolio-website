## 🛠️ Technical Details

### How Themes Work

Themes are defined using CSS variables that control:

- Background colors
- Text colors
- Border colors
- Accent colors
- Shadow colors
- Hover states
- Focus states
- Selection colors

### Creating Custom Themes

Want to create your own theme? It's easy!

1. Navigate to `lib/themes/definitions/`
2. Copy an existing theme file
3. Modify the color values
4. Add your theme to `lib/themes/definitions/index.ts`
5. Your theme appears in the admin panel!

Example theme structure:

```typescript
export const myTheme = {
  name: "My Theme",
  colors: {
    primary: "#your-color",
    secondary: "#your-color",
    accent: "#your-color",
    background: "#your-color",
    text: "#your-color",
  },
};
```
